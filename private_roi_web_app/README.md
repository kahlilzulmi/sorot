# ROI Web Application

Web-based advertisement eye tracking analysis tool with Flask + Vue.js.

## Features
- Smooth HTML5 video playback
- Interactive ROI editor
- Real-time gaze recording via WebSocket
- OBS auto-control and YouTube download
- Post-processing analysis

## Installation

```bash
pip install -r ../../requirements_roi_webapp.txt
```

## Main Files
- `video_roi_webapp.py` - Main Flask application
- `video_roi_demo.py` - Demo/test version
- `test_roi_gaze_connection.py` - Connection testing
- `test_roi_gaze_webapp.py` - Web app testing
- `test_vue.html` - Vue.js component testing

## Folders
- `static/` - CSS, JavaScript, and other static assets
- `templates/` - HTML templates
