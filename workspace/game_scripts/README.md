# Game Scripts

Eye-controlled game implementations with recording capabilities.

## Installation

```bash
pip install -r ../../requirements_workspace.txt
```

## Scripts

- `game3_with_recording.py` - Math quiz game with eye tracking recording
- `genvidsim4.py` - Video stimulus simulator
- `generated_positions.py` - Position generation utilities

## Usage
These scripts create interactive eye-controlled games and stimulus presentations for research purposes.
