# Analysis Scripts

Tools for analyzing, comparing, and visualizing eye tracking data.

## Installation

```bash
pip install -r ../../requirements_workspace.txt
```

## Scripts

- `comparison_pipeline.py` - Compare detection results with ground truth
- `generate_plots_from_csv.py` - Generate plots from CSV data
- `hitung_game_session.py` - Analyze game session data
- `offset_adjuster_gui.py` - GUI tool for adjusting time offsets

## Usage
These scripts help process and analyze the output from detection algorithms and game sessions.
